# ctci
Implementations of data structures and algorithms from Cracking the Coding Interview

## Install
Clone the repo
```
git clone https://github.com/kbougy/ctci.git
```

Install npm dependencies
```
npm install
```

## Test
Run the test suite
```
npm test
```

## Tasks
Build and bundle
```
gulp
```

Generate documentation
```
gulp typedoc
```

Linting
```
gulp tslint
```
